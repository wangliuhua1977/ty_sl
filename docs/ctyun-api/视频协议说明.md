## **视频协议说明**

### **转码流HLS协议简介**

   HLS (HTTP Live Streaming)，是Apple的动态码率自适应技术。主要用于PC和Apple终端的音视频服务，常见的应用还有微信。HLS包括一个m3u8的索引文件，TS媒体分片文件。天翼视联支持以m3u8链接的形式播放设备的实时视频。

### **转码流RTMP协议简介**

   RTMP(Real Time Messaging Protocol)实时消息传送协议，是Adobe Systems公司为Flash播放器和服务器之间音频、视频和数据传输开发的开放协议。天翼视联支持以rtmp链接的形式播放设备的实时视频。注意：RTMP依赖Flash Player

### **转码流WebRTC协议简介**

基于WebRTC协议的直播视频流是为了提升天翼视联用户在PC浏览器和手机浏览器直播场景下的视频播放体验，基于行业上更为先进的视频直播传输协议提供的一种新的直播视频流格式。相比原有的HLS格式，WebRTC能有效降低直播延迟、提升出流速度 、保障视频流安全、同时也具备跨平台等特点。

## **为什么这样设计？**

-   AppId + AppSecret（XXTEA）：适用于请求-响应模式，加密轻量、性能高。
-   RSA非对称加密：适用于平台→合作方的消息推送，确保只有持有私钥的您才能读取敏感告警。
