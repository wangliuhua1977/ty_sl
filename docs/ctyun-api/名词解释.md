## **名词解析**

名词

描述

门户

天翼视联-应用管理平台；在未登录状态，可查看对接指引，功能介绍，相关文档等信息；登录后，可查看当前登录用户授权的应用信息，管理员权限可以修改应用配置信息，如IP白名单，跨域域名等等

AppID/AppSecret

应用密钥对，平台（门户）创建接入应用后默认生成，主要用于接口认证时使用，6

访问令牌

accessToken,除了登录认证相关接口，其他能力接口都需要传accessToken访问令牌

RSA

RSA算法，公私钥在门户创建应用时自动生成，用于加解密部分敏感数据，用户可在门户查看解密密钥，具体参考接口文档

XXTea

XXTEA算法，对称加密算法，密钥为AppSecret，门户创建应用是自动生成。用于加解密部分敏感数据，具体参考接口文档

enterpriseUser

企业主账号手机号码，开通汇聚费销售品套餐的账号，只有用户无感知授权登录需要传

HMAC-SHA256

使用SHA-256生成哈希值的HMAC算法

云眼

天翼云眼客户端,如天翼云眼手机app，PC的BS云眼管理后台等

企业主

开通汇聚费销售品套餐的账号，设备收编（绑定级联）汇聚的云眼账号

设备主

指设备绑定的云眼普通账号

业务树（目录树）

只有企业主有业务树，由用户自定义设置层级关系，当企业主绑定或者级联新设备时，会将该设备添加到该企业主的业务mp树目录下

设备树

设备主所绑定的设备， 六级目录，前5级为行政区域信息，最后一级为设备

行业树（固定业务树）

行业树（固定业务树）由省份的天翼云眼管理后台分配的设备目录结构，主要目的是分管同一种行业的设备，以行业对设备树进行管理，使得设备的归属更加清晰和规范，如，天翼应急，平安慧眼，明厨亮灶等的行业设备，

拓展-渠道经理

指省份对接人：是该项目所在区域对应的区域经理（即拓展-渠道经理）
