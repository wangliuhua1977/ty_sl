### **通信协议**

-   平台采用 HTTPS 协议跟第三方应用交互，只支持 POST 请求方式，返回结果为 JSON 格式。
-   所有的请求和响应数据编码皆为 UTF-8 格式，URL 里的所有参数名和参数值请做 URL 编码。请求的 Content-Type请使用 application/x-www-form-urlencoded，则 HTTP Body体里的所有参数值也做 URL 编码。

使用appSecret 对所有传入参数(除signature)采用 XXTea 加密：params = XXTea((参数1=值&参数2=值&...),AppSecret)，请求参数无须排序。
