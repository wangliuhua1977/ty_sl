设备告警消息列表查询

### **1、设备告警消息列表查询**

字段标识

getDeviceAlarmMessage

接口描述

通过设备码获取设备告警信息列表

承载协议

HTTPS

承载网络

公网

请求方式

POST

数据格式

请求参数以 form 表单方式提交（application/x-www-form-urlencoded），响应结果为 JSON 格式

约束

无

接口URL

https://vcp.21cn.com/open/token/device/getDeviceAlarmMessage

业务方向

接入方→能力开放
