查询监控目录树

### **1、查询监控目录树**

字段标识

getReginWithGroupList

接口描述

获取当前账号所分配的区域，及其所有下级区域列表，此接口只是获取当前区域目录层级的下级目录结构，若需要获取当前目录下所有子级目录接口，需通过响应参数的hasChildren判读是否有下级目录并通过regionId递归全目录；若需要获取当前目录下的设备，可通过响应参数的havDevice判断当前目录是否有设备并通过获取监控目录设备列表相关接口获取设备列表（只有云眼的企业主（开通汇聚服务）用户才有权限调这个接口，普通用户调这个接口会报（50004：获取业务树目录列表异常）

承载协议

HTTPS

承载网络

公网

请求方式

POST

数据格式

请求参数以 form 表单方式提交（application/x-www-form-urlencoded），响应结果为 JSON 格式

约束

无

接口URL

https://vcp.21cn.com/open/token/device/getReginWithGroupList

业务方向

接入方→能力开放

请求参数

字段标识
