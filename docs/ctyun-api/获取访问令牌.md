# 天翼视联-能力开放门户

## **能力开放标准应用获取accessToken流程说明（企业主无感知鉴权兑换令牌）**

   调用能力开放平台原子能力需要先获取访问令牌（accessToken），当前文档描述能力开放门户创建的应用通过**用户无感知的方式获取访问令牌**。

**一、用户无感知方式获取能力开放平台访问令牌**

   适用场景：对接前已明确拉流企业主且由项目方签署信息安全函授权直接调用企业主名下设备场景（授权方式需勾选“用户无感知获取令牌”的标准API应用）。

   指引说明：

   （1）应用创建前上传访问令牌申请函件，描述项目的背景以及承诺拿到能力开放颁发的访问令牌后不将令牌用于非法和不当用途。

       1、能力开放平台应用访问令牌（accessToken有效期7天、refreshToken有效期30天）

       2、对接文档 （见下文 “getAccessToken” 接口描述）

```javascript
注意：访问令牌具有时间限制，当前方式取得的accessToken有效期为7天，当accessToken过期后请通过该接口刷新访问令牌，每次
刷新后refreshToken有效时间也会延长。       
```
