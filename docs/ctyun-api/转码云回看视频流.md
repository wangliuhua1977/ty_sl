获取云回看文件列表

### **2、获取云回看文件列表**

注：云回看文件列表查询的文件范围，和设备时间绑定的云存套餐相关，如7天云回看套餐即查询范围为7天，30天云回看套餐则为30天；若绑定的是普通云回看套餐，则云存会默认不间断保存回看视频，每个云回看文件一般时长为5分钟，若绑定的是事件云回看套餐，则云回看文件只有在触发对应事件是才会保存视频，一般是1分钟；

云存文件格式说明

云存储视频的封装格式为 MPEG-2 Program Stream，封装要求符合【GBT 28181-2016 公共安全视频监控互联网信息传输、交换、控制技术要求】中附录 C.1“基于 RTP 的视音频数据 PS 封装”

视频编码

1、H264/AVC，2、H265/HEVC

音视频编码

1、 pcm\_alaw, fITU-T G.711 A-law，2、aac, ISO/IEC 13818-7 Audio with ADTS transport

采样率

8000Hz

采样深度

16

声道

单声道(mono)
